# Hospital Readmission Analysis (Excel Project)

## Project Overview

This project focuses on analyzing hospital admission data to understand the key factors that lead to patient readmissions. The main objective is to identify patterns in patient behavior and medical conditions so that hospitals can make better decisions to improve patient care and reduce avoidable readmissions.

---

## Data Cleaning

- Missing values were handled carefully to ensure data accuracy  
- A new column `readmitted_flag` was created to clearly identify readmission cases  
- Inconsistent and invalid records were removed  
- The dataset was cleaned and structured properly for analysis  

---

## Feature Engineering

- Categorical data was transformed into a usable format  
- Readmission rates were calculated using averages  
- Data was grouped and summarized using Pivot Tables  
- Meaningful groupings were created to derive insights  

---

## Analysis Performed

- Readmission rate by Age  
- Readmission rate by Gender  
- Readmission rate by Race  
- Emergency visits vs Readmission  
- Diabetes impact on readmission  
- Medication change vs readmission  
- Inpatient visits vs readmission  

---

## Dashboard

An interactive Excel dashboard was created to present insights in a clear and user-friendly way. It helps in understanding:

- Readmission trends across age groups and demographics  
- Impact of emergency visits and inpatient history  
- Overall readmission rate and contributing factors  

---

## Key Insights

- Older patients tend to have higher readmission rates  
- Frequent emergency visits increase the likelihood of readmission  
- Chronic conditions such as diabetes contribute to higher readmission  
- Previous inpatient visits strongly indicate future readmission risk
