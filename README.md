# 🏥 Hospital Readmission Analysis (Excel Project)

## 📊 Project Overview

This project analyzes hospital admission data to identify patterns and factors that contribute to patient readmissions. The goal is to improve patient care and reduce unnecessary readmissions.

---

## 🧹 Data Cleaning

* Handled missing values
* Created a new column: `readmitted_flag`
* Removed inconsistent and invalid data
* Structured dataset for analysis

---

## ⚙️ Feature Engineering

* Converted categorical values into usable format
* Calculated readmission rates using average
* Grouped data using Pivot Tables

---

## 📊 Analysis Performed

* Readmission rate by Age
* Readmission rate by Gender
* Readmission rate by Race
* Emergency visits vs Readmission
* Diabetes impact on readmission
* Medication change vs readmission
* Inpatient visits vs readmission

---

## 📈 Dashboard

An interactive Excel dashboard was created to visualize key insights such as:

* Readmission trends by age and demographics
* Impact of emergency and inpatient visits
* Overall readmission rate

---

## 🔍 Key Insights

* Older patients have higher readmission rates
* Frequent emergency visits increase readmission risk
* Chronic conditions lead to longer hospital stays
* Previous inpatient visits strongly predict readmissions
