Hospital Readmission Analysis 
Project Overview

This project focuses on analyzing hospital admission data to understand the key factors that lead to patient readmissions. The main objective is to identify patterns in patient behavior and medical conditions so that hospitals can take better decisions to improve patient care and reduce avoidable readmissions.

Data Cleaning
Missing values were handled carefully to ensure data accuracy
A new column readmitted_flag was created to clearly identify readmission cases
Inconsistent and invalid records were removed
The dataset was cleaned and structured properly for analysis
Feature Engineering
Categorical data was transformed into a usable format
Readmission rates were calculated using averages
Data was grouped and summarized using Pivot Tables
New insights were derived by creating meaningful groupings
Analysis Performed
Readmission rate by Age
Readmission rate by Gender
Readmission rate by Race
Emergency visits vs Readmission
Diabetes impact on readmission
Medication change vs Readmission
Inpatient visits vs Readmission
Dashboard

An interactive Excel dashboard was created to present the insights in a clear and user-friendly way. It allows easy understanding of patterns such as:

Readmission trends across different age groups and demographics
Impact of emergency visits and inpatient history
Overall readmission rate and key contributing factors
Key Insights
Older patients tend to have higher readmission rates
Patients with frequent emergency visits are more likely to be readmitted
Chronic conditions such as diabetes increase the chances of readmission
Patients with prior inpatient visits show a strong likelihood of returning

Conclusion

Through this project, I gained a strong understanding of how raw healthcare data can be transformed into meaningful insights using Excel. Working with Pivot Tables helped me summarize large datasets efficiently and uncover patterns that are not immediately visible.

Creating the dashboard improved my ability to present data in a clear and structured way. I focused on making the visuals simple, readable, and interactive so that anyone can quickly understand the key findings.
Overall, this project helped me build a solid foundation in data analysis, improved my Excel skills, and gave me practical experience in turning data into actionable insights.
